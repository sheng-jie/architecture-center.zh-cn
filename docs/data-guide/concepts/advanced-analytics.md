# <a name="advanced-analytics"></a><span data-ttu-id="3263d-102">高级分析</span><span class="sxs-lookup"><span data-stu-id="3263d-102">Advanced Analytics</span></span>

<span data-ttu-id="3263d-103">高级分析超出了传统商业智能 (BI) 的历史报告和数据聚合范畴，它使用数学、概率和统计建模技术来实现预测处理和自动决策。</span><span class="sxs-lookup"><span data-stu-id="3263d-103">Advanced analytics goes beyond the historical reporting and data aggregation of traditional business intelligence (BI), and uses mathematical, probabilistic, and statistical modeling techniques to enable predictive processing and automated decision making.</span></span>

<span data-ttu-id="3263d-104">高级分析解决方案通常涉及以下工作负荷：</span><span class="sxs-lookup"><span data-stu-id="3263d-104">Advanced analytics solutions typically involve the following workloads:</span></span>

* <span data-ttu-id="3263d-105">交互式数据探索和可视化</span><span class="sxs-lookup"><span data-stu-id="3263d-105">Interactive data exploration and visualization</span></span>
* <span data-ttu-id="3263d-106">机器学习模型训练</span><span class="sxs-lookup"><span data-stu-id="3263d-106">Machine Learning model training</span></span>
* <span data-ttu-id="3263d-107">实时或批量预测处理</span><span class="sxs-lookup"><span data-stu-id="3263d-107">Real-time or batch predictive processing</span></span>

<span data-ttu-id="3263d-108">大多数高级分析体系结构包括以下部分或所有组件：</span><span class="sxs-lookup"><span data-stu-id="3263d-108">Most advanced analytics architectures include some or all of the following components:</span></span>

* <span data-ttu-id="3263d-109">**数据存储**。</span><span class="sxs-lookup"><span data-stu-id="3263d-109">**Data storage**.</span></span> <span data-ttu-id="3263d-110">高级分析解决方案需要使用数据来训练机器学习模型。</span><span class="sxs-lookup"><span data-stu-id="3263d-110">Advanced analytics solutions require data to train machine learning models.</span></span> <span data-ttu-id="3263d-111">数据科学家通常需要探索数据，以识别其预测特征、以及数据与预测值之间的统计关系（称为标签）。</span><span class="sxs-lookup"><span data-stu-id="3263d-111">Data scientists typically need to explore the data to identify its predictive features and the statistical relationships between them and the values they predict (known as a label).</span></span> <span data-ttu-id="3263d-112">预测的标签可以是量化值，例如，某个事物在将来的货币价值，或者航班延迟的分钟数。</span><span class="sxs-lookup"><span data-stu-id="3263d-112">The predicted label can be a quantitative value, like the financial value of something in the future or the duration of a flight delay in minutes.</span></span> <span data-ttu-id="3263d-113">它也可能表示分类的类（例如“true”或“false”、“航班延迟”或“无航班延迟”），或者类似于“低风险”、“中等风险”或“高风险”的类别。</span><span class="sxs-lookup"><span data-stu-id="3263d-113">Or it might represent a categorical class, like "true" or "false," "flight delay" or "no flight delay," or categories like "low risk," "medium risk," or "high risk."</span></span>

* <span data-ttu-id="3263d-114">**批处理**。</span><span class="sxs-lookup"><span data-stu-id="3263d-114">**Batch processing**.</span></span> <span data-ttu-id="3263d-115">若要训练机器学习模型，通常需要处理大量的训练数据。</span><span class="sxs-lookup"><span data-stu-id="3263d-115">To train a machine learning model, you typically need to process a large volume of training data.</span></span> <span data-ttu-id="3263d-116">训练模型可能需要一段时间（几分钟到几小时）。</span><span class="sxs-lookup"><span data-stu-id="3263d-116">Training the model can take some time (on the order of minutes to hours).</span></span> <span data-ttu-id="3263d-117">可使用以 Python 或 R 等语言编写的脚本执行此训练，并且可以使用分布式处理平台（例如 HDInsight 或 Docker 容器中托管的 Apache Spark）扩展训练，以减少训练时间。</span><span class="sxs-lookup"><span data-stu-id="3263d-117">This training can be performed using scripts written in languages such as Python or R, and can be scaled out to reduce training time using distributed processing platforms like Apache Spark hosted in HDInsight or a Docker container.</span></span>

* <span data-ttu-id="3263d-118">**实时消息引入**。</span><span class="sxs-lookup"><span data-stu-id="3263d-118">**Real-time message ingestion**.</span></span> <span data-ttu-id="3263d-119">在生产环境中，许多高级分析工具可将实时数据流馈送到已发布为 Web 服务的预测模型。</span><span class="sxs-lookup"><span data-stu-id="3263d-119">In production, many advanced analytics feed real-time data streams to a predictive model that has been published as a web service.</span></span> <span data-ttu-id="3263d-120">传入的数据流通常在某种形式的队列中捕获，流处理引擎从此队列提取数据，并以近乎实时的速度对输入的数据应用预测。</span><span class="sxs-lookup"><span data-stu-id="3263d-120">The incoming data stream is typically captured in some form of queue and a stream processing engine pulls the data from this queue and applies the prediction to the input data in near real time.</span></span>  

* <span data-ttu-id="3263d-121">**流处理**。</span><span class="sxs-lookup"><span data-stu-id="3263d-121">**Stream processing**.</span></span> <span data-ttu-id="3263d-122">训练模型后，通常很快（几毫秒）就能针对给定的一组特征执行预测（或评分）。</span><span class="sxs-lookup"><span data-stu-id="3263d-122">Once you have a trained model, prediction (or scoring) is typically a very fast operation (on the order of milliseconds) for a given set of features.</span></span> <span data-ttu-id="3263d-123">捕获实时消息后，可将相关的特征值传递给预测服务，以生成预测的标签。</span><span class="sxs-lookup"><span data-stu-id="3263d-123">After capturing real-time messages, the relevant feature values can be passed to the predictive service to generate a predicted label.</span></span>

* <span data-ttu-id="3263d-124">**分析数据存储**。</span><span class="sxs-lookup"><span data-stu-id="3263d-124">**Analytical data store**.</span></span> <span data-ttu-id="3263d-125">在某些情况下，预测的标签值将写入分析数据存储，以供报告和将来的分析。</span><span class="sxs-lookup"><span data-stu-id="3263d-125">In some cases, the predicted label values are written to the analytical data store for reporting and future analysis.</span></span>

* <span data-ttu-id="3263d-126">**分析和报告**。</span><span class="sxs-lookup"><span data-stu-id="3263d-126">**Analysis and reporting**.</span></span> <span data-ttu-id="3263d-127">顾名思义，高级分析解决方案通常生成某种形式的报告或分析源，其中包括预测的数据值。</span><span class="sxs-lookup"><span data-stu-id="3263d-127">As the name suggests, advanced analytics solutions usually produce some sort of report or analytical feed that includes predicted data values.</span></span> <span data-ttu-id="3263d-128">预测的标签值往往用于填充实时仪表板。</span><span class="sxs-lookup"><span data-stu-id="3263d-128">Often, predicted label values are used to populate real-time dashboards.</span></span>

* <span data-ttu-id="3263d-129">**业务流程**。</span><span class="sxs-lookup"><span data-stu-id="3263d-129">**Orchestration**.</span></span> <span data-ttu-id="3263d-130">尽管初始的数据探索和建模是由数据科学家以交互方式执行的，但许多高级分析解决方案会定期使用新数据重新训练模型 &mdash; 不断改进模型的准确性。</span><span class="sxs-lookup"><span data-stu-id="3263d-130">Although the initial data exploration and modeling is performed interactively by data scientists, many advanced analytics solutions periodically re-train models with new data &mdash; continually refining the accuracy of the models.</span></span> <span data-ttu-id="3263d-131">可以使用协调的工作流将此重新训练自动化。</span><span class="sxs-lookup"><span data-stu-id="3263d-131">This retraining can be automated using an orchestrated workflow.</span></span>

## <a name="machine-learning"></a><span data-ttu-id="3263d-132">机器学习</span><span class="sxs-lookup"><span data-stu-id="3263d-132">Machine learning</span></span>
<span data-ttu-id="3263d-133">机器学习是用于训练预测模型的数学建模技术。</span><span class="sxs-lookup"><span data-stu-id="3263d-133">Machine learning is a mathematical modeling technique used to train a predictive model.</span></span> <span data-ttu-id="3263d-134">一般原理是对一个大型历史数据集应用统计算法，以揭示它包含的字段之间的关系。</span><span class="sxs-lookup"><span data-stu-id="3263d-134">The general principle is to apply a statistical algorithm to a large dataset of historical data to uncover relationships between the fields it contains.</span></span>

<span data-ttu-id="3263d-135">需要在训练模型之前充分探索和准备数据的数据科学家通常会执行机器学习建模。</span><span class="sxs-lookup"><span data-stu-id="3263d-135">Machine learning modeling is usually performed by data scientists, who need to thoroughly explore and prepare the data before training a model.</span></span> <span data-ttu-id="3263d-136">这种探索和准备工作往往涉及到大量的交互式数据分析和可视化 &mdash; 通常是在专门针对此任务设计的交互式工具和环境中使用 Python 和 R 等语言。</span><span class="sxs-lookup"><span data-stu-id="3263d-136">This exploration and preparation typically involves a great deal of interactive data analysis and visualization &mdash; usually using languages such as Python and R in interactive tools and environments that are specifically designed for this task.</span></span>

<span data-ttu-id="3263d-137">在某些情况下，可以使用 Microsoft 开发的、获取的训练数据随附的[预先训练模型](/machine-learning-server/install/microsoftml-install-pretrained-models)。</span><span class="sxs-lookup"><span data-stu-id="3263d-137">In some cases, you may be able to use [pretrained models](/machine-learning-server/install/microsoftml-install-pretrained-models) that come with training data obtained and developed by Microsoft.</span></span> <span data-ttu-id="3263d-138">预先训练模型的优势是可以立即评分和分类新内容，即使没有所需的训练数据、用于管理大型数据集或训练复杂模型的资源。</span><span class="sxs-lookup"><span data-stu-id="3263d-138">The advantage of pretrained models is that you can score and classify new content right away, even if you don't have the necessary training data, the resources to manage large datasets or to train complex models.</span></span>

<span data-ttu-id="3263d-139">机器学习有两大类别：</span><span class="sxs-lookup"><span data-stu-id="3263d-139">There are two broad categories of machine learning:</span></span>

* <span data-ttu-id="3263d-140">**监督式学习**。</span><span class="sxs-lookup"><span data-stu-id="3263d-140">**Supervised learning**.</span></span> <span data-ttu-id="3263d-141">监督式学习机器学习最常采用的方法。</span><span class="sxs-lookup"><span data-stu-id="3263d-141">Supervised learning is the most common approach taken by machine learning.</span></span> <span data-ttu-id="3263d-142">在监督式学习模型中，源数据包括一组特征数据字段，这些字段与一个或多个标签数据字段存在数学关系。</span><span class="sxs-lookup"><span data-stu-id="3263d-142">In a supervised learning model, the source data consists of a set of *feature* data fields that have a mathematical relationship with one or more *label* data fields.</span></span> <span data-ttu-id="3263d-143">在机器学习的训练阶段，数据集包含特征和已知标签，将应用某种算法来拟合某个函数，以便处理特征，计算相应的标签预测。</span><span class="sxs-lookup"><span data-stu-id="3263d-143">During the training phase of the machine learning process, the data set includes both features and known labels, and an algorithm is applied to fit a function that operates on the features to calculate the corresponding label predictions.</span></span> <span data-ttu-id="3263d-144">通常，一部分训练数据集将会保留，用于验证训练后的模型的性能。</span><span class="sxs-lookup"><span data-stu-id="3263d-144">Typically, a subset of the training dataset is held back and used to validate the performance of the trained model.</span></span> <span data-ttu-id="3263d-145">训练模型后，可将它部署到生产环境，用于预测未知值。</span><span class="sxs-lookup"><span data-stu-id="3263d-145">Once the model has been trained, it can be deployed into production, and used to predict unknown values.</span></span> 

* <span data-ttu-id="3263d-146">**非监督式学习**。</span><span class="sxs-lookup"><span data-stu-id="3263d-146">**Unsupervised learning**.</span></span> <span data-ttu-id="3263d-147">在非监督式学习模型中，训练数据不包括已知的标签值。</span><span class="sxs-lookup"><span data-stu-id="3263d-147">In an unsupervised learning model, the training data does not include known label values.</span></span> <span data-ttu-id="3263d-148">算法将会根据它对数据的第一次接触来生成预测。</span><span class="sxs-lookup"><span data-stu-id="3263d-148">Instead, the algorithm makes its predictions based on its first exposure to the data.</span></span> <span data-ttu-id="3263d-149">最常见形式的非监督式学习是“聚集”，其中的算法确定如何最适当地根据特征中的统计类似性，将数据拆分为指定数量的聚集。</span><span class="sxs-lookup"><span data-stu-id="3263d-149">The most common form of unsupervised learning is *clustering*, where the algorithm determines the best way to split the data into a specified number of clusters based on statistical similarities in the features.</span></span> <span data-ttu-id="3263d-150">在聚集中，预测结果是输入特征所属的聚集数。</span><span class="sxs-lookup"><span data-stu-id="3263d-150">In clustering, the predicted outcome is the cluster number to which the input features belong.</span></span> <span data-ttu-id="3263d-151">尽管有时可以使用它们直接生成有用的预测，例如，使用聚集来客户数据库中的用户组，但非监督式学习方法更常用于识别哪些数据最有利于在训练模型时提供给监督式学习算法。</span><span class="sxs-lookup"><span data-stu-id="3263d-151">While they can sometimes be used directly to generate useful predictions, such as using clustering to identify groups of users in a database of customers, unsupervised learning approaches are more often used to identify which data is most useful to provide to a supervised learning algorithm in training a model.</span></span>

<span data-ttu-id="3263d-156">基于数学技术（例如线性回归或逻辑回归）的机器学习模型已经面世了一段时间。</span><span class="sxs-lookup"><span data-stu-id="3263d-156">Machine learning models based on mathematical techniques like linear or logistic regression have been available for some time.</span></span> <span data-ttu-id="3263d-157">最近，基于神经网络的深度学习技术越来越流行。</span><span class="sxs-lookup"><span data-stu-id="3263d-157">More recently, the use of *deep learning* techniques based on neural networks has increased.</span></span> <span data-ttu-id="3263d-158">这股潮流在一定程度上是由高度可伸缩的处理系统推动的，它们可以减少训练复杂模型的时间。</span><span class="sxs-lookup"><span data-stu-id="3263d-158">This is driven partly by the availability of highly scalable processing systems that reduce how long it takes to train complex models.</span></span> <span data-ttu-id="3263d-159">此外，大数据的日益流行使得我们可以在各种领域更轻松地训练深度学习模型。</span><span class="sxs-lookup"><span data-stu-id="3263d-159">Also, the increased prevalence of big data makes it easier to train deep learning models in a variety of domains.</span></span>

## <a name="model-deployment"></a><span data-ttu-id="3263d-170">模型部署</span><span class="sxs-lookup"><span data-stu-id="3263d-170">Model deployment</span></span>

<span data-ttu-id="3263d-171">支持 AI 应用程序的预测服务可以利用自定义机器学习模型，或可用于访问预先训练模型的现成认知服务。</span><span class="sxs-lookup"><span data-stu-id="3263d-171">The predictive services that support AI applications may leverage custom machine learning models, or off-the-shelf cognitive services that provide access to pretrained models.</span></span> <span data-ttu-id="3263d-172">将自定义模型部署到生产环境的过程称为操作化，其中，在处理环境中经过训练和测试的相同 AI 模型被序列化，可供外部应用程序和服务进行批处理或自助预测。</span><span class="sxs-lookup"><span data-stu-id="3263d-172">The process of deploying custom models into production is known as operationalization, where the same AI models that are trained and tested within the processing environment are serialized and made available to external applications and services for batch or self-service predictions.</span></span> <span data-ttu-id="3263d-173">若要使用模型的预测功能，可以首先使用相同的机器学习库（包含用于训练模型的算法）来反序列化和加载该模型。</span><span class="sxs-lookup"><span data-stu-id="3263d-173">To use the predictive capability of the model, it is deserialized and loaded using the same machine learning library that contains the algorithm that was used to train the model in the first place.</span></span> <span data-ttu-id="3263d-174">此库提供预测函数（通常称为评分或预测），这些函数提取模型和特征作为输入，并返回预测。</span><span class="sxs-lookup"><span data-stu-id="3263d-174">This library provides predictive functions (often called score or predict) that take the model and features as input and return the prediction.</span></span> <span data-ttu-id="3263d-175">然后，将此逻辑包装在可由应用程序直接调用或可公开为 Web 服务的函数中。</span><span class="sxs-lookup"><span data-stu-id="3263d-175">This logic is then wrapped in a function that an application can call directly or can be exposed as a web service.</span></span>
