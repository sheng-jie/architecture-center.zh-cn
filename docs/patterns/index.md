# <a name="cloud-design-patterns"></a>云设计模式

这些设计模式可用于在云中构建可靠且可缩放的安全应用程序。

每种模式描述了该模式解决的问题、有关应用该模式的注意事项，以及基于 Microsoft Azure 的示例。 大多数模式都包含了代码示例或代码片段，演示如何在 Azure 中实现该模式。 但是，无论托管在 Azure 还是其他云平台中，大多数模式都与任一分布式系统相关。
